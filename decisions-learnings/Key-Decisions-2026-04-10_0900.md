## Learnings

**L-037: Instagram and YouTube block WebFetch — JS-only rendering**
Instagram (instagram.com) and YouTube (youtube.com) return only JavaScript configuration code when fetched via WebFetch — no readable content. Follower counts, post content, and engagement metrics are inaccessible via WebFetch. For social media audits: use Facebook (accessible), LinkedIn (accessible via company pages), and direct mobile/desktop access for Instagram and YouTube.

**L-038: Facebook company pages return JS on direct fetch — use search results instead**
Direct WebFetch to facebook.com/PremierRealtyKe returns raw CSS/JS. Facebook likes count (~6,814) was recovered via a Google search result snippet, not via direct fetch. When auditing Facebook: search Google for the page name first to recover like count from search result metadata.

**L-039: Kenya real estate ranking sites are the best competitive intelligence source**
Gazebo Homes (gazebohomes.com), Finsco Africa (finscoafrica.com), Hao Finder (haofinder.com), and Tuko.co.ke all publish searchable, fetchable top-company lists with descriptions. These are fast, high-quality sources for Kenya real estate competitive landscape research — significantly faster than trying to scrape individual company sites.

**L-040: BuyRentKenya agent profiles are a reliable listing volume proxy**
buyrentkenya.com/estate-agent/{slug} returns a clean, fetchable page with listing count, property types, geographic coverage, and professional credentials (KPDA, EARB membership). This is an underutilized research source for Kenya real estate company profiling — more informative than the company's own website in many cases.
