## Learnings

**L-035: Always audit existing platform performance before prescribing social media strategy**
- **What happened:** The first proposal was built on the assumption that LinkedIn and Instagram were the correct primary platforms — a reasonable default for a professional in 2026. User corrected this: Facebook is actually Keziah's highest engagement platform.
- **Why it matters:** A proposal built on the wrong platform priority would waste Keziah's time building on platforms she struggles with while neglecting the one that already works.
- **How to apply:** Before finalizing any social media strategy, ask: "Which platform currently gets you the most engagement or responses?" and "Where do your existing clients or contacts tend to reach out?" Use the answer to anchor the strategy.

**L-036: docx permission conflict when overwriting open workspace files**
- **What happened:** Attempted to `cp` a revised .docx over an existing file in the workspace folder — received "Permission denied / Operation not permitted." The file was open or locked.
- **Fix:** Save revised file under a new versioned filename (e.g., `_v2.docx`) rather than overwriting. Inform user which version is current.
- **How to apply:** Always version docx output files (`_v1`, `_v2`) when iterating in the same session. Never attempt to overwrite.
