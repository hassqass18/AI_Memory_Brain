## 2. Site Configuration — 4-Tower Complex

> **CONFIRMED (2026-04-11):** The development is a **3-tower residential complex** on Plot MN/I/1691. Benjamin and Franklin are two wings of one combined tower (Benjamin-Franklin Tower). Total: **88 units**.

### 2.1 Tower Summary

| Tower | Unit Type | Units | Unit Size (sqft / sqm) | Built-Up Area (sqft / sqm) |
|-------|-----------|------:|------------------------:|----------------------------:|
| **Lincoln Tower** | 3 Bedroom | 22 | 2,617.58 sqft / 243.2 sqm | 59,278 sqft / 5,508 sqm |
| **Washington Tower** | 3 Bedroom | 22 | 2,617.58 sqft / 243.2 sqm | 59,278 sqft / 5,508 sqm |
| **Benjamin-Franklin Tower** | 2 Bedroom (Benjamin Wing) + 1 Bedroom (Franklin Wing) | 44 (22+22) | 1,311.18 / 966.44 sqft | 54,652 sqft / 5,078 sqm |
| **Total** | | **88** | | **173,208 sqft / 16,093 sqm** |

> **Note:** Benjamin-Franklin Tower is one combined building with two wings. The Benjamin Wing houses 22 × 2BR units (1,311.18 sqft / 121.8 sqm). The Franklin Wing houses 22 × 1BR units (966.44 sqft / 89.8 sqm). Lincoln and Washington are standalone 3BR towers. Total: 88 units.

### 2.2 Tower Floor Configuration

Each tower/wing runs from **1st to 11th floor** (11 floors), with **2 units per floor**.
