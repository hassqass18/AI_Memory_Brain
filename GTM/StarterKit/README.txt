AI MEMORY BRAIN STARTER KIT
===========================

This folder contains the 3 core templates for the AI Memory Brain Starter Kit - a complete system for building persistent, portable AI memory that works across any AI tool (Claude, ChatGPT, etc.).

FILES INCLUDED:
===============

1. AMB-Starter-Master-Context-Template.docx
   - The "brain spine" - loads everything your AI needs to know
   - Sections: Business Overview, Key Stakeholders, Active Projects, Operating Standards, Agent Roster, Session History Index
   - Save as: Master-AI-Context.md
   - Use: Load at the start of every session
   - Size: ~5 pages

2. AMB-Starter-Session-Open-Template.docx
   - The handshake protocol - initializes your AI with full context
   - Includes: Load context, state goals, review last session, load project context, confirm understanding
   - Ready-to-use template with examples
   - Customization guide for different session types (Strategy, GTM, Technical, Operations)
   - Size: ~5 pages

3. AMB-Starter-Session-Close-Template.docx
   - The debrief and memory save - captures what you accomplished
   - Includes: Session summary, key decisions, files created, open items, next steps, Master Context updates
   - Ready-to-use template with examples
   - Session type variations and best practices
   - Size: ~5 pages

QUICK START:
=============

Step 1: Fill in the Master Context template with your business information
Step 2: Save it as Master-AI-Context.md in your project directory
Step 3: At the start of each session, use the Session Open template to load context
Step 4: At the end of each session, use the Session Close template to save state
Step 5: Update your Master Context with new decisions and progress

TOTAL SETUP TIME: ~30 minutes
