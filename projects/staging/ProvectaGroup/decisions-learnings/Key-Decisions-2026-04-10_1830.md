## Learnings

- **Zoho CRM API v2 does NOT support workflow rule creation** — returns `API_NOT_SUPPORTED` with note that v9 is required. Must use Settings > Automation > Workflow Rules in the UI.
- **Zoho Campaigns API requires its own OAuth token** — the CRM token scopes (`ZohoCRM.*`) do not grant Campaigns access. Need to generate a new OAuth token with `ZohoCampaigns.*` scope to automate mailing list and sequence creation.
- **Zoho CRM textarea field limit** — the Leads module has a maximum number of custom textarea fields. The 31st textarea (`Additional_Context`) hit this limit and was created as `text` type instead. This is functionally equivalent for short notes.
- **`Current_CRM` and `Audit_Type` already existed** — from the original setup-zoho.js run. The new script handled duplicates cleanly with skip logic.
- **Field API names in Zoho CRM** — Zoho sometimes appends `__c` to custom field API names. The implementation script checks both variants when detecting existing fields.
