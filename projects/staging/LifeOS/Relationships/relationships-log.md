## How to Log

During daily check-in, Claude appends a dated block per interaction. Format:

```
## [YYYY-MM-DD]
**Contact:** [name]
**Type:** [Call / Message / In-Person / Meeting / Email]
**Quality:** [1–5]
**Topics:** [brief — what did you talk about?]
**Takeaways:** [what you learned, commitments made]
**Action Items:** [anything you need to do as a result]
**Trust Account:** [Deposit / Neutral / Withdrawal] — [brief why]
**Next Touch:** [when to follow up]
---
[repeat block for each person]
```

One block per person per day. Skip days with no meaningful interactions.
