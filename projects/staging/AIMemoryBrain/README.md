## Product Overview

**Problem:** Every AI session starts from zero. Project knowledge, decisions, architectural context, and failure modes must be manually re-explained every time — or are silently lost.

**Solution:** A three-tier, lazy-loading, git-backed memory system. Any AI model loads full project context in 60 seconds. Every session, decision, and risk is indexed and searchable.

**Differentiators:**
- Model-agnostic (Claude, ChatGPT, Gemini, any LLM)
- No special tooling required (Markdown + Git)
- Scales across unlimited concurrent projects
- Organizational memory as a side effect
