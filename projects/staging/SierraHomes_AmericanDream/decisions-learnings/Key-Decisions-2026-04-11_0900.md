## Learnings

- **Git lock files at session open** — HEAD.lock, ORIG_HEAD.lock, refs/heads/main.lock, objects/maintenance.lock were present and owned by host OS process; cannot be removed from sandbox. Files are current via mount. Known pattern — proceed without git pull when locks are present.

- **openpyxl: FORMAT_NUMBER_COMMA_SEP1 not importable** — removed in newer versions of openpyxl. Define custom number format strings directly instead (e.g., `'#,##0;(#,##0);"-"'`). Do not import from `openpyxl.styles.numbers`.

- **lbl_font() italic keyword** — custom font helper functions must include all parameters used. If `italic` is needed, either add it as a kwarg to the helper or use `Font(name="Arial", ...)` directly.

- **Excel Sales Commission circular reference** — commission = 3% × revenue; revenue includes commission-funded transactions. Resolved by computing commission as 3% of total gross revenue (no circularity at the line-item level; commission is an outflow, not included in revenue).
